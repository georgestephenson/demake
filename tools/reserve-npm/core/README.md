# @demake/core

**This name is reserved.** `@demake/core` is the conversion engine behind
[`demake`](https://www.npmjs.com/package/demake): the deterministic image
pipeline, console specs, and codegen that turn any image into hardware-compliant
art for 8/16-bit-era consoles and handhelds.

This is a placeholder release that holds the name while the project is built in
the open. The real engine is on the way.
